pandas → Para manejar y analizar datos financieros estructurados (transacciones, presupuestos, etc.).
numpy → Para realizar cálculos matemáticos avanzados.
scipy → Para cálculos de optimización financiera.
finance o pyfinance → Para análisis de datos financieros.
QuantLib → Si necesitas cálculos más avanzados de tasas de interés o inversiones.
dateutil → Para manejo de fechas en transacciones.
finsim → Para simulaciones de ahorro e inversión.
pyportfolioopt → Para optimizar inversiones y portafolios financieros.
matplotlib → Para gráficos simples como barras, líneas, circulares.
seaborn → Para gráficos estadísticos más avanzados.
plotly → Para gráficos interactivos y dashboards.
